# Job Application Tracker

A simple React + TypeScript application for tracking job applications. This tool allows users to add, filter, and export job applications, providing a clean and efficient way to manage the job search process.

## Features

- **Add Job Applications**: Input company name, position title, job offer link, application date, status, and notes.
- **Filter Applications**: Filter job applications by status (Applied, Interviewing, Offer, Rejected).
- **Export Data**: Export your applications as CSV or JSON files.
- **Local Storage Persistence**: Data is saved in localStorage, so applications remain even after closing the browser.
- **Responsive Design**: Clean and responsive UI using Tailwind CSS.

## Technologies Used

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **PapaParse** (for CSV export)

## Setup Instructions

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/job-application-tracker.git
   cd job-application-tracker
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Run the Development Server:**

   ```bash
   npm run dev
   ```

4. **Open the App:**

   Visit [http://localhost:5173](http://localhost:5173) in your browser.

## Exporting Data

- **Export as CSV:** Click the "Export as CSV" button to download your job applications in CSV format.
- **Export as JSON:** Click the "Export as JSON" button to download your job applications in JSON format.
